## 1. 发布边界

本次是开发、自用和受控测试用途的 Windows 预览快照，不是正式 GA 发行：

- 根 `VERSION`、应用内部版本和 Git 预览标签统一为 `0.1.0-preview.1` / `v0.1.0-preview.1`。
- 只交付 x64 NSIS 安装包，不建立自动更新服务，不作稳定兼容承诺。
- 安装包暂不签名，可能触发 Windows SmartScreen 未知发布者提示。
- 角色名称、角色素材和应用图标仍待公开发行权利审查；未通过前不公开推广或商业分发。
- M16 在本次 tag 后仍保持进行中，正式 `v0.1.0` 需要完成升级/回滚、低端性能、许可和公开发行检查。

## 5. tag 门槛与回滚

只有自动化检查、NSIS 构建和上述真实安装关键路径通过后，才执行：

```powershell
npm run release:tag
git push origin v0.1.0-preview.1
```

预览版没有自动更新器。回滚方式是：退出绨络、备份应用数据目录、卸载当前包并安装上一预览包；如果设置 schema 已升级，不承诺旧版能够读取新版数据，必要时同步恢复对应版本的数据备份。

## 6. 未关闭风险

- 角色 IP、名称、图标和衍生素材尚未完成公开发行授权审查。
- 依赖许可证与 notice 尚需在公开发行前形成完整、可随包交付的清单。
- 安装包没有代码签名；低端硬件、ARM64、早期 Windows 10 和远程桌面未形成性能/兼容基线。
- 应用标识符沿用早期模板值是有意兼容决策；正式版前若要更名，必须先实现数据目录和 DPAPI 密钥迁移。

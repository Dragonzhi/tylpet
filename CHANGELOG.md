# 变更日志

本项目从 `0.1.0-preview.1` 开始记录面向使用者的版本变化。版本号遵循语义化版本；`preview` 表示功能和数据格式仍可能调整，不构成稳定兼容承诺。

## [0.1.0-preview.1] - 待发布

### 新增

- 分层 SVG 桌宠、鼠标跟随、待机动画、拖动惯性、招手、语音口型和命名动作运行时。
- 透明窗口、自定义拖动、轮廓点击穿透、原生菜单、托盘和可靠退出路径。
- 可恢复番茄钟、系统音乐状态反应和可中断 Windows 本机 TTS。
- 离线 Mock 与 OpenAI-compatible 流式对话、受控白名单 Agent 工具和工具时间线。
- 用户可管理的长期记忆、确定性羁绊和对话式记忆候选确认。
- 版本化观察事件、受限本机插件桥接、开发 Agent hook 示例和插件 SDK。
- 独立 SVG Animation Studio 与生产角色 rig/motions 工作流。

### 安全与隐私

- API key 通过 Windows DPAPI 加密，外部模型通信默认要求 HTTPS。
- Agent 不获得 shell、文件、进程、任意网络、DOM 或任意原生命令能力。
- 系统媒体观察不读取音频内容或媒体元数据；插件事件经过授权、校验、限频和脱敏诊断。

### 已知限制

- 安装包尚未签名，也没有自动更新器。
- 本版本为非官方同人技术预览；角色素材公开发行权利审查尚未完成。
- 仅以 Windows 10 22H2 / Windows 11 x64 为目标，其他平台与架构未承诺兼容。
